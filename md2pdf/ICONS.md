# Generating Extension Icons

The extension requires three icon files:
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)  
- `icon128.png` (128x128 pixels)

## Option 1: Use the HTML Generator

1. Open `icons/generate-icons.html` in your browser
2. Click each button to generate and download the icons
3. Save them in the `icons/` directory with the correct names

## Option 2: Create Your Own Icons

You can create custom icons using any image editor. The icons should:
- Be square PNG images
- Represent the markdown-to-PDF conversion functionality
- Have good contrast for visibility at small sizes

## Option 3: Use Online Tools

Use online icon generators or favicon generators that support multiple sizes.
